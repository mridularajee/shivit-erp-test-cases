# SHIVITT ERP – Manual Test Cases

##  Project Overview
This repository contains manual test cases for the **SHIVITT ERP application**.  
The test cases are written and maintained in Excel format for functional validation of ERP modules.

---

##  Modules Covered
- Item Master
- Mandatory Field Validation
- Input Field Validation
- File Upload Validation
- Save & Notification Validation

---

##  Type of Testing
- Manual Testing
- Functional Testing
- UI Validation
- Field Validation

---

##  Tools Used
- Microsoft Excel
- GitHub (for version control)

---

##  Repository Structure
- `shivitt-erp-test-cases.xlsx` – Contains detailed test cases with steps, expected results, actual results, and screenshots.

---

##  Test Case Fields Included
- Test Case ID
- Test Scenario
- Test Steps
- Expected Result
- Actual Result
- Status (Pass/Fail)
- Screenshot Reference
- Comments
